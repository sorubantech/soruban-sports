# Soruban Sports — Cricket Academy SaaS Platform

## Product Overview

**Date:** 2026-03-16
**Status:** Brainstorming Phase

---

## Vision

A SaaS platform for cricket academies to manage day-to-day operations with modern AI technology. Think of it as an **autonomous college system for cricket** — academies select from a common library + their own material, combine them, and deliver structured coaching with full visibility for students and parents.

## Core Concept: The Autonomous College Model

| Education Concept | Cricket Academy Equivalent |
|-------------------|--------------------------|
| University | Our SaaS Platform (Soruban Sports) |
| College | Individual Academy |
| Syllabus Board | Common material library (platform-provided animated videos, drills) |
| College-specific electives | Academy's own custom materials |
| Semesters | Stages / Levels |
| Subjects | Skill categories (Batting, Bowling, Fielding, Wicketkeeping, Fitness) |
| Exams | Coach-designed assessments |
| Marks / GPA | Player performance score |
| Degree | Stage completion certificate |

## User Roles

| Role | Description |
|------|-------------|
| **Platform Admin** | Soruban Sports team — manages common library, platform settings |
| **Academy Owner** | Registers academy, manages finances, views dashboards |
| **Head Coach** | Designs curriculum, creates exams, oversees coaching quality |
| **Assistant Coach / Staff** | Marks attendance, guides students, conducts drills |
| **Student** | Logs in, views stage progress, accesses learning material, takes exams |
| **Parent** | Views child's progress, attendance, assessment scores |

---

## Key User Flows

### Student / Parent Journey
1. Login → Choose academy (from enrolled academies on platform)
2. Admission process with the academy
3. Onboarded → Assigned role (Batsman / Bowler / Allrounder / Wicketkeeper)
4. Placed in a stage based on initial assessment
5. Access stage-wise material: animated videos, drills, practice routines
6. Guided by staff based on role and current stage
7. Takes exams → Gets scores → Progresses to next stage

### Academy Owner Journey
1. Register academy on platform
2. Build curriculum: Common library + Own material + Combined
3. Configure operations: attendance method, batch timings, fee structure
4. Monitor: dashboards for student progress, revenue, coach performance

## Confirmed Feature Categories

1. **Academy Management** — Registration, multi-branch, configurable attendance
2. **Curriculum Builder** — Common + custom material, stage-wise organization
3. **Student Lifecycle** — Admission, role assignment, stage progression
4. **Content Library (Platform Asset)** — Animated videos for every shot, expert-recommended
5. **Assessment System** — Coach-created exams with cricket patterns, scoring
6. **AI Features** — Video-based technique analysis, automated performance reports
7. **Progress Dashboard** — Student/parent view with historical tracking
8. **Workload & Injury Management** — Ball count tracking, overtraining alerts
9. **Comparative Benchmarking** — Cross-academy anonymized performance data
10. **Coach Marketplace** — Profiles, credentials, hiring through platform
11. **Gamification** — Badges, leaderboards, engagement features
12. **Smart Scheduling** — AI-optimized batches, weather-based rescheduling
13. **Fee Management** — Online payments, reminders, revenue dashboards

## Current Market Pain Points We Solve

| Pain Point | Our Solution |
|------------|-------------|
| No structured data | Everything digitized — curriculum, attendance, scores |
| No coach accountability | Performance tracking per coach |
| Revenue leakage | Automated fee tracking, payment reminders |
| No parent visibility | Real-time progress dashboards |
| Can't scale to multiple branches | Multi-branch management from day one |
| Subjective evaluations | Data-backed assessments and progression |
| No talent pipeline | Stage completion + benchmarking = selection readiness |
