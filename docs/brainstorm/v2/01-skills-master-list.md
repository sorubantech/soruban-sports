## Overview

Complete list of **112 skills** across **7 categories** that the platform tracks, teaches, and assesses. This is the expanded V2 list — adds 24 new skills over V1 (88), including a brand-new **Nutrition & Recovery** category.

> **Academy Freedom:** Academies can enable/disable any skill, add custom skills, rename categories, and reorder everything. This is the platform's default master list.

## Skill Tagging System

Every skill in the platform is tagged with metadata for filtering, searching, and curriculum building:

```
Skill: Cover Drive
├── ID: B11
├── Category: Batting
├── Sub-Category: Front Foot Attack
├── Stage Introduced: Stage 3
├── Difficulty: Intermediate
├── Player Role Relevance:
│   ├── Batsman: Essential
│   ├── Allrounder: Essential
│   ├── Bowler: Optional
│   └── Wicketkeeper: Essential
├── Format Relevance:
│   ├── Test: High
│   ├── ODI: High
│   └── T20: Medium
├── Prerequisites: [B2 Stance, B4 Footwork, B10 Straight Drive]
├── Related Skills: [B12 On Drive, B13 Off Drive, B14 Square Drive]
├── Content: [Technique Video, Common Mistakes, Drill Video, Assessment Card]
└── Assessment Criteria: "Drive through cover region 10/20 balls"
```

## Skill Dependencies (Prerequisite Map)

```
Stage 1: Grip → Stance → Backlift → Footwork
                ↓
Stage 2: Forward Defensive → Back Foot Defensive → Leaving
                ↓                    ↓
Stage 3: Straight Drive          Pull Shot
         Cover Drive              Square Cut
         On Drive                 Dead Bat Defense
                ↓
Stage 4: Sweep → Late Cut → Back Foot Punch
                ↓
Stage 5: Reverse Sweep → Dancing Down Track → Hook Shot → Lofted Shots
                ↓
Stage 6: Switch Hit → Scoop → Ramp (T20 Mastery)
                ↓
Stage 7: All shots under pressure, format-specific mastery
```

> Each arrow means "should be learned before." Academies can override prerequisites.
