## Overview

This document covers how **practice sessions are planned, tracked, and connected to skill development**. The goal: every minute of practice is linked to measurable skill growth.

## 1. Practice Session Planner

### What It Is

Coaches build structured session plans from a library of drills, each linked to specific skills. The session becomes a trackable unit — not just "2 hours of nets."

### Session Template Structure

```
┌──────────────────────────────────────────────────────┐
│  SESSION PLAN                                         │
│  Batch: Evening U-14  |  Date: 2026-03-22 (Sat)      │
│  Coach: Ravi Kumar    |  Duration: 2 hours             │
│  Focus: Front Foot Batting + Agility                  │
├──────────────────────────────────────────────────────┤
│                                                        │
│  1. WARM-UP (15 min)                                   │
│     ├── Dynamic stretching            [FIT4, FIT8]     │
│     └── Agility ladder + cone drills  [FIT5]           │
│                                                        │
│  2. SKILL FOCUS — BATTING (40 min)                     │
│     ├── Cover Drive drill             [B11]            │
│     │   └── 15 balls each, focus on footwork            │
│     ├── On Drive drill                [B12]            │
│     │   └── 15 balls each, weight transfer              │
│     └── Straight Drive revision       [B10]            │
│         └── 10 balls each, quick refresher              │
│                                                        │
│  3. MATCH SIMULATION (30 min)                          │
│     ├── Scenario: 30 runs needed off 5 overs           │
│     │   Skills tested: [MG3, MG4, B32]                 │
│     └── Rotate 3 batsmen, rest field                   │
│                                                        │
│  4. FIELDING (20 min)                                  │
│     ├── High catches (10 each)        [F1]             │
│     └── Flat throw accuracy           [F10]            │

```

### How Coaches Create Sessions

**Option A — Build from Scratch**
1. Select batch and date
2. Add blocks (Warm-up, Skill Focus, Match Sim, Fielding, Fitness, Cool-down)
3. For each block, select drills from the drill library
4. System auto-tags which skills are covered
5. Save as one-off or recurring template

**Option B — Use Templates**
Platform provides ready-made session templates:

| Template | Focus | Duration | Skills Covered |
|----------|-------|----------|---------------|
| Batting Foundations | Grip, Stance, Forward Def | 90 min | B1-B4, B6, FIT4 |
| Fast Bowling Basics | Run-up, Action, Line & Length | 90 min | BW1-BW5, FIT1, FIT8 |
| Match Day Prep | Scenario batting, fielding drills | 120 min | B32, MG3, MG4, F1-F4 |
| Fitness Friday | Speed, agility, endurance | 60 min | FIT1-FIT8 |
| Spin Masterclass | Off-break, leg-break, sweep play | 120 min | BW21, BW24, B22-B25 |
| Keeper Special | Stance, taking pace/spin, stumping | 90 min | WK1-WK5 |
| Recovery Session | Light drills, stretching, video review | 60 min | FIT4, FIT8, NR3, MG10 |

**Option C — Copy & Modify**
Copy a previous session, adjust drills for the day.

### Session History

Every session is logged:
- Date, batch, coach, students present (attendance linked)
- Skills practiced (auto-tracked from session plan)
- Coach notes (free text observations per student)
- Duration breakdown (how much time on each category)

## 2. Drill Library

### What It Is

A searchable library of drills, each linked to one or more skills. Platform provides default drills; academies add their own.

### Drill Card Format

```
┌─────────────────────────────────────────────┐
│  DRILL: Cone Target Cover Drive             │
│  Skill: B11 (Cover Drive)                   │
│  Category: Batting > Front Foot Attack      │
│  Difficulty: ★★☆ Intermediate               │
│  Duration: 10-15 min                         │
│  Players: 1 batsman + 1 feeder              │
│  Equipment: Bat, 6 balls, 4 cones           │
├─────────────────────────────────────────────┤
│  SETUP:                                      │
│  Place 4 cones in cover region (2m apart)    │
│  Feeder throws from 12m, full length,        │
│  outside off stump                           │
│                                              │
│  EXECUTION:                                  │
│  1. Batsman plays cover drive                │
│  2. Ball must pass between cones             │
│  3. 20 balls per set                         │
│                                              │
│  SUCCESS CRITERIA:                           │
│  ✅ Basic: 8/20 through cones               │
│  ✅ Good: 12/20 through cones               │
│  ✅ Excellent: 16/20 through cones          │
│                                              │
│  COACH WATCH POINTS:                         │
│  - Front foot reaching pitch of ball         │
│  - Head over front knee                      │
│  - Full face of bat towards target           │
│  - Follow-through towards cover              │
│                                              │
│  VARIATIONS:                                 │
│  - Easier: Underarm throw, wider target      │
│  - Harder: Bowling machine at 100kph         │
│  - Advanced: Random line (cover OR straight)  │
│                                              │
│  VIDEO: [Platform technique video B11]       │
│  SOURCE: Platform / Custom (academy upload)  │
└─────────────────────────────────────────────┘
```

### Drill Difficulty Levels

Each skill has drills at 3 levels, mapped to progression:

| Level | Description | Typical Stage | Method |
|-------|-------------|--------------|--------|
| **Basic** | Simplified, controlled environment | Stage 1-2 | Underarm throws, stationary, wide targets |
| **Intermediate** | Semi-realistic, some pressure | Stage 3-4 | Bowling machine, tighter targets, time pressure |
| **Advanced** | Match-realistic, full pressure | Stage 5-7 | Live bowling, fielders present, match scenarios |

### Who Sees What

| Role | View |
|------|------|
| **Coach** | All students in their batch — aggregate + individual exposure |
| **Head Coach** | All batches — identify imbalanced training across the academy |
| **Student** | Own exposure data — "I need more fielding practice" |
| **Parent** | Child's exposure — "My child is getting enough bowling practice?" |
| **Academy Owner** | Academy-wide — are coaches following balanced training plans? |

### Imbalance Alerts

System generates alerts when:
- A skill category has 0 hours for 2+ weeks
- One category exceeds 60% of total practice time
- A student's exposure is significantly below batch average
- Key skills for the student's role haven't been practiced (e.g., WK student with no keeping practice)

## 4. Bowler Workload Management

### Why This Matters

Fast bowlers, especially juniors, are highly injury-prone. Overuse leads to stress fractures. Academies need a workload tracker.

### ECB/ICC Youth Bowling Guidelines (Built into Platform)

| Age Group | Max Balls/Day | Max Days/Week | Mandatory Rest |
|-----------|--------------|--------------|---------------|
| Under 13 | 36 balls (6 overs) | 3 days | 1 day between bowling days |
| Under 15 | 42 balls (7 overs) | 3 days | 1 day between bowling days |
| Under 17 | 48 balls (8 overs) | 4 days | 1 day after 2 consecutive days |
| Under 19 | 54 balls (9 overs) | 4 days | 1 day after 2 consecutive days |
| Senior | No limit (coach discretion) | 5 days | 1 day after 3 consecutive days |

> Academies can customize these limits. Platform provides defaults based on ICC/ECB guidelines.

### Workload Dashboard (Coach View)

```

│  BOWLER WORKLOAD — Evening U-14 Batch — This Week    │
├──────────────────────────────────────────────────────┤
│                                                        │

│  ──────────────  ───  ───  ───  ───  ───  ─────  ───── │
│  Arjun S.         -   36    -   30    -    66    126   │
│  Status: ✅ OK                                         │
│                                                        │
│  Pradeep R.      42    -   42    -   36   120    126   │
│  Status: ⚠️ APPROACHING LIMIT (95%)                   │
│                                                        │
│  Karthik M.      42   36   42   36    -   156    126   │
│  Status: 🔴 EXCEEDED LIMIT — Rest required!           │
│                                                        │
│  Vikram N.        -    -   24    -    -    24    126   │
│  Status: ✅ OK                                         │
│                                                        │
│  ALERTS:                                               │

│  ⚠️ Pradeep at 95% weekly limit — max 6 balls tomorrow│
└──────────────────────────────────────────────────────┘
```

### How It Works

1. **Logging**: Coach enters ball count per bowler after each session (or session plan auto-calculates based on drill specifications)
2. **Tracking**: System accumulates daily, weekly, and monthly ball counts
3. **Alerts**: Automatic notifications when bowler approaches or exceeds limits
4. **History**: Weekly/monthly bowling load graphs — track trends over time
5. **Rest Enforcement**: System flags consecutive bowling days and recommends rest

### Match Day Integration

When scoring a match, the system also counts overs bowled. This feeds into the workload tracker automatically.

## 5. Personal Development Plan (PDP)

### What It Is

A coach-set, student-visible plan of 3-5 focus areas per month. Gives students clear direction on what to improve.

### PDP Card

```
┌──────────────────────────────────────────────────────┐
│  PERSONAL DEVELOPMENT PLAN                            │
│  Student: Ravi Kumar  |  Role: Batsman                │
│  Stage: 3 (Developing) | Month: March 2026            │
│  Set by: Coach Suresh  | Review: March 31             │
├──────────────────────────────────────────────────────┤
│                                                        │
│  FOCUS AREA 1: Back Foot Play                         │
│  Skills: B17 (Square Cut), B19 (Pull Shot)            │
│  Current Score: Cut 55%, Pull 60%                     │
│  Target Score: Cut 70%, Pull 75%                      │
│  Drills: Short ball pull drill (3x/week)              │
│          Cut shot target drill (2x/week)              │
│  Progress: ████████░░░░ 65% → target 72.5%           │
│  Coach Note: "Footwork improving, needs to commit     │
│              to the pull earlier"                      │
│                                                        │
│  FOCUS AREA 2: Running Between Wickets                │
│  Skills: B32 (Running), B33 (Calling)                 │
│  Current Score: 50%                                   │
│  Target Score: 70%                                    │
│  Drills: 2-run timed drill (every session)            │
│  Progress: ██████░░░░░░ 55% → target 70%             │
│  Coach Note: "Calling is the main issue, not speed"   │
│                                                        │
│  FOCUS AREA 3: Fitness — Speed                        │
│  Skills: FIT1 (Speed)                                 │
│  Current Score: 58%                                   │
│  Target Score: 70%                                    │
│  Drills: Sprint intervals (2x/week), shuttle runs     │
│  Progress: ████████░░░░ 62% → target 70%             │
│                                                        │
│  OVERALL PDP PROGRESS: 61% → target 71%              │
│  Status: 🔄 On Track                                  │
│  Next Review: March 31 — Coach will update targets    │
└──────────────────────────────────────────────────────┘
```

### PDP Workflow

```
Coach creates PDP (start of month)
    ↓
Student sees focus areas in app — "What should I work on"
    ↓
Sessions include PDP-linked drills (coach plans around PDP)
    ↓
Scores update from assessments and drill performance
    ↓
Mid-month: Coach adds observations/notes
    ↓
End of month: Coach reviews PDP
    ├── Met targets? → Celebrate, set new PDP for next month
    └── Not met? → Extend focus, adjust drills, add coach notes
    ↓
Parent sees monthly PDP progress in their app
```

### PDP History

All PDPs are stored chronologically, creating a development journey:

```
March 2026: Back foot play, Running, Speed → 85% achieved
February 2026: Cover drive, On drive, Agility → 90% achieved
January 2026: Forward defence, Stance, Flexibility → 100% achieved
```

## 6. Practice Types & Categories

### Session Types (Coach Selects When Creating)

| Type | Description | Skills Focus |
|------|-------------|-------------|
| **Net Session** | Standard batting/bowling in nets | Batting, Bowling |
| **Fielding Session** | Dedicated fielding and catching practice | Fielding |
| **Fitness Session** | Physical conditioning, strength, cardio | Fitness |
| **Match Simulation** | Simulated game scenarios with scoring | Mental/Game, All |
| **Video Review** | Watching and analyzing footage | Mental/Game, Self-Analysis |
| **Recovery Session** | Light drills, stretching, rest | Fitness, Nutrition/Recovery |
| **Assessment Day** | Formal skill testing and evaluation | All (based on test) |
| **Specialist Session** | Role-specific (keeper drills, spin coaching) | Role-specific |
| **Open Nets** | Unstructured practice, student-led | Student choice |
| **Theory/Classroom** | Rules, strategy, nutrition education | Mental/Game, Nutrition |

### Weekly Session Template (Recommended)

For a typical academy running 6 days/week:

```
Monday    → Net Session (Batting focus)
Tuesday   → Net Session (Bowling focus)
Wednesday → Fielding + Fitness
Thursday  → Net Session (Mixed batting/bowling)
Friday    → Match Simulation
Saturday  → Assessment / Open Nets
Sunday    → REST
```

> Academy sets their own schedule. Platform provides templates as starting points.

│                                                        │
│  SESSION RATING: ★★★★☆ (Good)                         │
│                                                        │
│  HIGHLIGHTS:                                           │
│  + Arjun's cover drive has improved significantly      │
│  + Pradeep bowled consistent line all session           │
│                                                        │
│  CONCERNS:                                             │
│  - Karthik's bowling action seems strained — monitor   │
│  - Vikram not focusing during fielding drills           │
│                                                        │
│  INDIVIDUAL NOTES (optional — per student):            │
│  Arjun S.: Cover drive excellent, move to On Drive     │
│  Karthik M.: Check bowling load — arm seems tired      │
│  Meera R.: First time attempting pull shot — promising  │
│                                                        │

│  Focus on on drive (B12) + back foot defence (B7)     │
│  Add fielding catch drill for weaker catchers           │

### Report Features
- **Quick entry**: Pre-filled from session plan, just add observations
- **Student tagging**: Tag specific students for positive or concern notes
- **Photo/Video**: Attach clips from the session
- **Private**: Only visible to staff (Head Coach, other coaches, admin) — not students/parents
- **Handoff**: If another coach takes the next session, they see this report

## 8. Integration with Existing Features

### Session → Attendance
- Session plan auto-creates attendance sheet
- Students marked present/absent feed into exposure tracking

### Session → Skills → Progress
- Skills practiced in session flow into student's skill exposure
- Assessment scores during session update progress dashboard

### Session → PDP
- System highlights which PDP focus skills were covered
- Coach sees: "Ravi's PDP needs pull shot practice — not done in 2 weeks"

### Session → Bowler Workload
- Bowling drills auto-count toward weekly ball limits
- Alert before session if a bowler is near their limit

### Session → Social Feed
- Coach can share session highlights to academy feed
- Auto-generated: "Today's U-14 session focused on cover drive — 12 students attended"

### Session → Parent View
- Parents see: session attended, skills practiced, coach notes (non-private summary only)
