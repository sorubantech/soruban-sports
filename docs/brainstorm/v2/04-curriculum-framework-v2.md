## What Changed in V2

This document captures the **expanded curriculum structure** with 113 skills (up from 88), the new Nutrition & Recovery category, and enhanced stage-wise progression for ALL skill categories — not just batting.

V1 had detailed stage-wise breakdown only for batting. V2 adds stage-wise progression for **Bowling, Fielding, Wicket-Keeping, Fitness, Mental/Game Sense, and Nutrition/Recovery**.

> For the complete skills master list, see `v2/01-skills-master-list.md`
> For training/practice module, see `v2/02-training-practice-module.md`
> For performance/growth features, see `v2/03-performance-growth-features.md`

## 1. Stage Structure (Unchanged from V1)

| Stage | Name | Age Guide | Focus |
|-------|------|-----------|-------|
| **Stage 1** | Foundation | 6-8 yrs | Basic grip, stance, hand-eye, fun drills |
| **Stage 2** | Beginner | 8-10 yrs | Basic technique, catching, throwing |
| **Stage 3** | Developing | 10-12 yrs | Shot selection, bowling variations, fielding positions |
| **Stage 4** | Intermediate | 12-14 yrs | Match awareness, advanced shots, spin/pace basics |
| **Stage 5** | Advanced | 14-16 yrs | Role specialization, match strategy, fitness |
| **Stage 6** | Elite | 16-18 yrs | Tournament readiness, mental conditioning |
| **Stage 7** | Professional Prep | 18+ yrs | First-class readiness, selection trials |

> Age is a guideline. Progression is skill-based, not age-based.

## 5. Stage-Wise Fitness Progression (NEW in V2)

| Stage | Focus | Benchmarks (age-adjusted) |
|-------|-------|--------------------------|
| **Stage 1** | FIT1 Speed, FIT2 Stamina, FIT4 Flexibility, FIT5 Agility, FIT8 Injury Prevention | Fun games (tag, relay), basic stretching, 200m run |
| **Stage 2** | + FIT3 Strength, FIT6 Core, FIT7 Reaction Time | Shuttle runs, basic core exercises, reaction ball drills |
| **Stage 3** | All areas — increasing intensity | 400m timed run, plank 30sec, reaction test baseline |
| **Stage 4** | Age-appropriate strength training begins | 800m run, push-ups/sit-ups targets, agility course time |
| **Stage 5** | Structured fitness regime | 1600m run, gym introduction, flexibility benchmarks |
| **Stage 6** | Sport-specific conditioning | Beep test level targets, weight training, HIIT |
| **Stage 7** | Professional-level fitness | Yo-yo test, comprehensive strength benchmarks |

### Fitness Assessment Template

Every stage has a fitness test with age-appropriate benchmarks:

```
FITNESS TEST — Stage 3 (10-12 yrs)
├── Sprint: 30m in < 5.5 seconds
├── Endurance: 400m in < 1:45
├── Core: Plank hold 30 seconds
├── Agility: Shuttle run (10m x 4) in < 12 seconds
├── Flexibility: Sit and reach — touch toes
├── Reaction: Catch 6/10 reaction ball drops
├── Strength: 10 push-ups, 15 sit-ups
└── Injury Prevention: Demonstrates warm-up routine correctly
```

## 6. Stage-Wise Mental / Game Sense Progression (NEW in V2)

| Stage | Skills Introduced | Teaching Method |
|-------|------------------|----------------|
| **Stage 1** | MG7 Teamwork & Communication | Team games, group drills, celebrating others |
| **Stage 2** | MG1 Concentration, MG5 Confidence, MG9 Goal Setting | Focus drills (count balls), positive reinforcement, simple weekly goals |
| **Stage 3** | MG3 Shot Selection, MG6 Resilience | Net scenarios ("play 10 balls without getting out"), handling dismissal positively |
| **Stage 4** | MG2 Pressure Handling, MG4 Match Awareness, MG10 Self-Analysis, MG11 Adaptability | Match simulations with scoreboard pressure, watching own video clips, adjusting to conditions |
| **Stage 5** | MG8 Tactical Planning, MG12 Leadership/Captaincy | Setting fields, bowling plans, vice-captaincy in internal matches |
| **Stage 6** | All areas under tournament pressure | Tournament preparation, mental conditioning workshops, sports psychology basics |
| **Stage 7** | Professional mental readiness | Selection trial simulation, media handling, pressure scenarios |

### Mental/Game Sense Assessment

Unlike physical skills, mental skills are assessed through:
1. **Match simulation scoring** — Decision quality in scenarios
2. **Coach observation** — Behavioral assessment over time
3. **Self-assessment questionnaire** — Student rates own mental game
4. **Video review** — Reviewing decisions made in matches

## 7. Stage-Wise Nutrition & Recovery Progression (NEW in V2)

| Stage | Topics | Teaching Method |
|-------|--------|----------------|
| **Stage 1** | NR1 Hydration, NR5 Injury Log | "Drink water!" reminders, reporting any pain to coach |
| **Stage 2** | + NR2 Pre-Training Nutrition, NR3 Post-Training Recovery | Simple food guidelines, cool-down routine introduction |
| **Stage 3** | + NR4 Sleep & Rest | Sleep hours tracking (optional), rest day importance |
| **Stage 4** | All areas — deeper understanding | Nutrition plans, pre/post-match eating, recovery protocols |
| **Stage 5+** | Sport science basics | Protein timing, hydration science, sleep optimization, injury prevention programs |

### Nutrition Content Types

| Type | Description |
|------|-------------|
| **Guidelines Card** | Simple do/don't cards (e.g., "Eat 2 hours before practice") |
| **Meal Plan Templates** | Sample meal plans for match day, training day, rest day |
| **Hydration Calculator** | Based on session duration + temperature → water intake target |
| **Recovery Checklist** | Post-session checklist: stretch, hydrate, protein, rest |
| **Injury Report Form** | Student/coach reports pain/injury → logs to student profile |

## 10. V2 Decisions & Open Questions

### Confirmed Decisions (carried from V1)
1. 7 stages, fully configurable by academy
2. Skill-based progression, not age-based
3. Coach must approve stage promotion
4. No automated placement — coach places on admission
5. No cross-academy portability
6. Digital certificates on stage completion

### New Decisions Needed for V2

| # | Question | Recommendation |
|---|----------|---------------|
| 1 | Add Nutrition & Recovery as a formal category? | Yes — especially important for junior athletes |
| 2 | Should fitness have formal stage-wise assessments like batting? | Yes — with age-appropriate benchmarks |
| 3 | Mental/Game Sense — formal assessment or coach observation only? | Hybrid — coach observation (70%) + scenario quiz (30%) |
| 4 | Bowling workload tracking — MVP or Phase 2? | MVP — injury prevention is critical, and it's a differentiator |
| 5 | Practice session planner — structured or free-form at MVP? | Start with free-form (coach writes notes). Phase 2: structured planner |
| 6 | How many drill difficulties per skill? | 3 levels (basic, intermediate, advanced) |
| 7 | Should students self-track fitness outside academy? | Phase 2 — self-logging for runs, exercises done at home |
