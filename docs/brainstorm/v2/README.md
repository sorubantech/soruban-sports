# Version 2 — Skills, Training & Performance Growth

**Date:** 2026-03-21
**Status:** Brainstorming / Open for Discussion

---

## What's New in V2

V2 expands the platform from a **management tool** to a **performance engine**. Key additions:

1. **113 skills** (up from 88) across 7 categories including new **Nutrition & Recovery**
2. **Practice session planner** — structured, skill-linked sessions with drill library
3. **Skill exposure tracking** — automatically know how much time each student spends on each category
4. **Bowler workload management** — injury prevention through ball-count tracking
5. **Personal Development Plans (PDP)** — coach-set monthly focus areas
6. **Skill radar chart** — visual spider graph on every profile
7. **Achievement badges** — gamification with 30+ earnable badges
8. **Video progress diary** — side-by-side improvement clips
9. **Anonymous peer comparison** — motivating without toxicity
10. **Stage-wise progression for ALL categories** — not just batting
